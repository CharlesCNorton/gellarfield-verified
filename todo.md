1. `FieldState` inductive: `Active | Flickering | Collapsed`. `WarpCondition`: `Calm | Storm | Rift`. `DaemonClass`: `Lesser | Greater | Primordial`. `ShipZone` as finite enumeration with adjacency relation.
2. `field_strength : nat -> Q` with axiomatized decay: `strength(t+1) = strength(t) - drain(warp_condition(t))` where `drain Calm = 1/100`, `drain Storm = 1/10`, `drain Rift = 1/3`.
3. State thresholds: `strength > 3/4 -> Active`, `1/4 < strength <= 3/4 -> Flickering`, `strength <= 1/4 -> Collapsed`. Prove these partition `[0,1]` exhaustively.
4. Generator model: `fuel : nat -> Q` strictly decreasing, `fuel(t+1) = fuel(t) - consumption(warp_condition(t))`. When `fuel(t) = 0`, `strength(t') = 0` for all `t' >= t`.
5. Flickering dynamics: during Flickering, strength oscillates via `osc(t) = amplitude * sin(t)` discretized, creating windows where effective strength dips below Collapsed threshold intermittently.
6. Navigator as `Option NavigatorState` where `NavigatorState` carries `sanity : Q` decaying under Rift exposure. Navigator alive iff `sanity > 0`.
7. Course correction: Navigator alive reduces warp transit duration by factor `correction_factor : Q` (axiomatized `0 < cf < 1`), directly reducing total fuel burn and exposure.
8. Navigator death theorem: `sanity <= 0 -> transit_duration = unbounded`. Formalize as: without Navigator, no upper bound on time to Materium translation, so fuel exhaustion is certain.
9. `incursion_possible` predicate gated on field state AND warp condition AND daemon class. Lesser needs Flickering+Storm or worse. Greater needs Collapsed or Rift. Primordial needs only Rift. Active+Calm blocks everything. Prove all 27 combinations decidable.
10. Zone breach: `breach : nat -> ShipZone -> Prop`. Hull zones breach first. `breach(t, z) /\ adjacent(z, z') -> breach(t + delay(bulkhead(z,z')), z')`. Bulkhead intact: `delay = 10`. Destroyed: `delay = 1`.
11. Crew as `list CrewMember` each carrying `exposure : nat -> Q` monotonically nondecreasing. `exposure(t+1) = exposure(t) + warp_flux(t)` when in breached zone with Collapsed field, `+0` otherwise.
12. Corruption threshold: `corrupted(c, t) <-> exposure(c, t) > threshold(c)`. Each crew member has individual threshold drawn from `[1, 10]`. Prove corruption is irreversible: `corrupted(c, t) -> corrupted(c, t')` for `t' >= t`.
13. **Collapse theorem**: if `field_strength` hits Collapsed during Storm and remains so for duration `d`, then `|{c | corrupted(c, d)}| / |crew| >= 1 - exp(-lambda * d)` where lambda depends on warp severity. As `d -> inf`, fraction -> 1. No vacuity — requires composing decay, breach propagation, and exposure accumulation.
14. **Flickering survival lemma**: if Flickering and generator restored (fuel injected) within `T_restore` steps, then for all crew `c`, `exposure(c, T_restore) < min_threshold(crew)`. Bounded survival.
15. **Bulkhead isolation theorem**: crew in zone `z` with all adjacent bulkheads intact, during Flickering of duration `d < delay`, have `exposure = 0`. Sealed zones buy real time.
16. **Fuel impossibility theorem**: for any initial fuel `F` and Warp Storm, exists `T` such that `fuel(T) = 0`. No finite fuel survives unbounded Storm transit. Straightforward from strict decrease but necessary foundation.
17. **Navigator necessity theorem**: compose fuel impossibility + Navigator death -> unbounded duration -> fuel exhaustion -> Collapsed -> collapse theorem. Chain of four lemmas, no single step trivial.
18. **Primordial penetration theorem**: during Rift, Primordial incursion is possible regardless of field state. Even Active fields fail. Prove `forall s, incursion_possible s Rift Primordial`.
19. **Calm transit safety theorem**: `warp_condition = Calm` throughout transit with sufficient fuel implies `field_strength` stays Active implies no incursion possible implies zero corruption. The happy path, formally.
20. **Monotonicity of doom**: `field_strength` is nonincreasing when no fuel is added. `exposure` is nondecreasing. `corruption_count` is nondecreasing. Three monotonicity lemmas composing into irreversibility of degradation.
21. **Storm escalation lemma**: `Storm` drains 10x faster than `Calm`. Prove that Storm transit of duration `T` produces equivalent degradation to Calm transit of `10T`. Quantified severity scaling.
22. **Breach cascade timing**: for a ship with `n` zones in a line, total time from hull breach to innermost zone breach is `sum of delays`. With intact bulkheads: `10n`. All destroyed: `n`. Closed-form bounds.
23. **Critical fuel threshold**: compute minimum fuel `F_min` such that for given warp condition and transit duration `T`, `fuel(T) > 0`. Decidable: `F_min = consumption * T`. Below this, collapse guaranteed.
24. **Grand composition**: combine all results into `warp_transit_outcome`: given initial fuel, crew, Navigator state, warp condition, and transit duration, classify outcome as `Safe | Survivable | Doomed` with proofs that classification is exhaustive and decidable.
